# Rhythmz Blocks Plugin

## A WordPress block-based plugin built on The Events calendar Pro

### Blocks
- Events Block
- Icons Block
- Three D Block
- Gallery Block
- Popup Block
- Venue Map Block

## Setup
  * Install dependencies: cd ./block && npm install
  * Build CMD: cd ./block && npm run build
  * Watch CMD: cd ./block && npm run watch
  * Start development server: cd ./block && npm run start
  * Build for production: cd ./block && npm run build
